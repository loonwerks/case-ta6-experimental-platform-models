This folder contains the Phase 2 AADL models where all SW threads (periodic) are scheduled to run on a single processer.
The AGREE contracts are updated and new contracts are created to model the "asynchronous" AADL/HAMR semantics.
The SW-level single layer proof completes in about 9 mins. Intel Core i7-9850H CPU @ 2.60 Ghz 2.59 Ghz RAM 32.0 GB
AGREE Analysis setting: SMT solver Z3, Timeout in seconds: 1000
